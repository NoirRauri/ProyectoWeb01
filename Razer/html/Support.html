High frequency FAQ

What is a mobile phone baseband?

Baseband is the inherent frequency band (frequency bandwidth) of the original electrical signal without modulation and transformation sent by the Baseband signal source (also called the transmitter). The "baseband" of a mobile phone has two meanings: one refers to the hardware, the communication module (modem) of the mobile phone and other equipment; the other refers to the software, which supports the control software (firmware) required by the communication module of the mobile phone and other equipment to complete signal processing. The query path of the baseband version information in the MIUI system: Settings---About Device---All specs---Baseband Version.The baseband version information is updated with the update of the MIUI version, and there may be cases where the MIUI system version is updated but the baseband version is not.

How to deal with the loss of baseband information on the mobile phone?

After the baseband information is lost, the SIM card will not be recognized. You can go to Settings---About Device---All specs---Baseband Version to check the baseband version. If the baseband version is displayed as "unknown" or "baseband information is lost", which may means that the baseband firmware is abnormal, you can try to update MIUI to the latest version to fix it; if it is still not resolved, apply for after-sales service.

Why does the baseband version change?

Baseband (software) refers to the control software (firmware) required to support the communication module of mobile phones and other devices to complete signal processing. This part of the firmware is stored in the mobile phone system software package (ROM package) and needs to be updated with the MIUI version to upgrade, but The baseband version exists independently, and there may be cases where the system version is upgraded but the baseband version is not upgraded.

Notes:

The baseband chip design at the hardware level relies on the CPU platform. Some CPU platforms will integrate the baseband chip, and some baseband processors will exist independently. If you use the backup software on the Xiaomi phone to perform backup, you failed to see the previous backup when restoring the backup, but there is a corresponding folder in the corresponding directory of the virtual SD card, which may be caused by the following reasons:
  • 1. The backup software will be improved with the system upgrade. If downgrades the phone system or the system version span is too large, the backup may not be able to restore.
  • 2. During the backup process or after the backup is completed, if an error occurs in a file in the backup, the backed-up file may not be detected.
Troubleshooting steps:
  • 1. If the system version span is too large and cannot be restored, it is recommended to contact the after-sales service.
  • 2. If the backup cannot be restored due to an error in a certain file during or after the backup, perform the following operations:
First, we call the original user backup as A backup, and create a new backup as B backup. Enter the MIUIBackupAllbackup directory of file management, and there will be folders for A backup and B backup. Enter the backup folder of A, and put the bak-format files that need to be restored to the folder corresponding to backup B. Then put the description.xml file in backup A in the corresponding folder of backup B. After the operation is complete, restart the phone and try to restore backup B to restore the contents of backup A.
Notes:
If the backup is encrypted, this method cannot solve the problem.
